# Wafer Map Pro 专业版模块

## 功能

- 标准 Bin 分类系统（10 种）
- 晶圆坐标系统和缺口显示
- 多模式显示（Bin Map、Heat Map）
- 专业统计分析
- 标准数据格式导出（CSV、SINF）

## 文件结构

```
wafer-map-pro/
├── index.vue                  # 专业版演示页面
├── components/
│   ├── WaferMapPro.vue       # 专业版组件
│   └── WaferMap.vue          # 基础组件
└── README.md
```
