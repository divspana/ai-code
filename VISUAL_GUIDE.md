## 🎬 操作流程动画

### 流程 1: 选择文件夹

```
1. 初始状态
   ┌─────────────┐
   │ 空白页面    │
   └─────────────┘
        ↓
2. 点击按钮
   ┌─────────────┐
   │ [选择文件夹]│ ← 点击
   └─────────────┘
        ↓
3. 系统对话框
   ┌─────────────────┐
   │ 📁 选择文件夹   │
   │                 │
   │ [取消] [选择]   │
   └─────────────────┘
        ↓
4. 显示文件树
   ┌─────────────────┐
   │ 📁 folder       │
   │   📄 file1      │
   │   📄 file2      │
   └─────────────────┘
```

### 流程 2: 上传文件

```
1. 点击上传
   [开始上传] ← 点击
        ↓
2. 队列处理
   ┌──────────────────┐
   │ 队列: [1,2,3,4,5]│
   └──────────────────┘
        ↓
3. 并发上传 (最多6个)
   ┌─────┬─────┬─────┬─────┬─────┬─────┐
   │ 文件1│ 文件2│ 文件3│ 文件4│ 文件5│ 文件6│
   │ ↑↑↑ │ ↑↑↑ │ ↑↑↑ │ ↑↑↑ │ ↑↑↑ │ ↑↑↑ │
   └─────┴─────┴─────┴─────┴─────┴─────┘
        ↓
4. 逐个完成
   ✓ 文件1 完成
   ✓ 文件2 完成
   ↑ 文件3 上传中
   ↑ 文件4 上传中
   ⏸ 文件5 等待中
   ⏸ 文件6 等待中
```

### 流程 3: 分片上传

```
大文件 (100MB)
        ↓
分片 (2MB/片)
        ↓
┌─────┬─────┬─────┬─────┬─────┐
│ 片1 │ 片2 │ 片3 │ ... │ 片50│
└─────┴─────┴─────┴─────┴─────┘
        ↓
逐片上传
  片1 ████ 100% ✓
  片2 ████ 100% ✓
  片3 ██▯▯  50% ↑
  片4 ▯▯▯▯   0% ⏸
  ...
        ↓
合并分片
  [合并中...]
        ↓
完成 ✓
```

## 🖼️ 实际效果预览

访问 `http://localhost:5173/folder-upload` 查看实际效果。

### 功能演示步骤

1. **选择文件夹**
   - 点击"选择文件夹"按钮
   - 选择一个包含多个文件的文件夹

2. **查看文件树**
   - 文件以树形结构展示
   - 可以展开/折叠文件夹
   - 查看文件信息

3. **开始上传**
   - 点击"开始上传"按钮
   - 观察进度条变化
   - 查看状态更新

4. **暂停/继续**
   - 上传过程中点击"暂停上传"
   - 再次点击"开始上传"继续

5. **查看统计**
   - 观察底部统计信息
   - 实时更新的数据

## 💡 使用技巧

### 1. 快速选择

- 使用键盘快捷键快速操作
- 双击文件夹展开/折叠

### 2. 批量操作

- 一次性选择整个文件夹
- 自动处理所有文件

### 3. 进度监控

- 实时查看上传进度
- 了解剩余时间

### 4. 错误处理

- 失败的文件会标红
- 可以单独重新上传
